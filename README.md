## **💪 기획 배경**
- 오픈 API인 pokeapi 를 이용하여, 포켓몬 도감을 만들며 웹 프론트에서 서버와 API 통신하는 방법을 학습하고자 하였습니다.
- CSS-in-JS(Emotion)를 다뤄보고 컴포넌트 기반 스타일링 방식에 대해 학습하고자 하였습니다.
- react-router-dom, infinity-scroll 등 여러 라이브러리를 학습하고자 하였습니다.

## **💪 주요 기능**
- 포켓몬 목록 (Poke API)
- 무한 스크롤 기능
- 포켓몬 상세 페이지 (Poke API)
- 페이지 라우팅 (메인 <-> 상세)
- 포켓몬 이미지 테마 변경
